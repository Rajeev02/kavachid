# @kavach-samples/react-native

This is an official sample demonstrating how to integrate the **Kavach SDK** and **Kavach Shield Engine (KSE)** into a react-native application.

## Overview
This sample showcases the 'Secure by Default, Configurable by Exception' paradigm. It demonstrates how to capture telemetry and handle dynamic biometric step-ups triggered by KSE.
